# AI-анализатор медицинских анализов
API-сервис на базе LLM, предназначенный для анализа результатов медицинских лабораторных исследований.
Система принимает структурированные данные анализов (например, показатели крови), выявляет отклонения от общепринятых референсных значений и формирует интерпретацию результатов в виде структурированного JSON-ответа.

Сервис может:
- обрабатывать входные данные анализов
- определять показатели, выходящие за пределы нормы
- генерировать краткое описание состояния на основе данных
- предлагать возможные причины отклонений (без постановки диагноза)
- формировать рекомендации общего характера
