# Подходы для оценки критериев AI-системы

## Таблица подходов к оценке

| Критерий                     | Тип оценки   | Подход                                   | Обоснование                                                        |
|------------------------------|--------------|------------------------------------------|--------------------------------------------------------------------|
| Фактическая согласованность  | Субъективная | AI-as-a-Judge + человеческая оценка      | Требует понимания медицинского контекста, возможны неоднозначности |
| Локальная согласованность    | Точная       | Функциональная корректность              | Можно напрямую сравнить ответ с входными данными                   |
| Полнота выявления отклонений | Точная       | Сравнение с эталоном (точное совпадение) | Есть ожидаемый список отклонений                                   |
| Безопасность                 | Смешанная    | Правила + AI-as-a-Judge                  | Часть проверяется правилами, часть - контекстно                    |
| Следование формату (JSON)    | Точная       | Функциональная корректность              | JSON можно валидировать автоматически                              |
| Задержка                     | Точная       | Прямое измерение                         | Измеряется в секундах                                              |
| Стоимость                    | Точная       | Прямое измерение                         | Рассчитывается по стоимости запроса                                |
